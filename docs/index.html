jsConvert

jsConvert is a simple JavaScript Library for Converting string and keys of object between cases (camelCase, snake_case, PascalCase, dot.case, path/case, text case, Sentence case, Header Case, UPPERCASE, lowercase).

Using for both Node.JS and Browser. Available on NPM.
